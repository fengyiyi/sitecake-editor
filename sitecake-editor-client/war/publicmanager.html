Toolbar

First of all SiteCake toolbar is dragable. So you can move it around the screen if it stands in your way. Most of the time toolbar will be empty, options will appear only when they are meaningful.

How To Edit?

Clicking on every content element let you edit it. Clicking outside quits edit mode. Just like that.

How To Add Content?

By draging Heading /Sub Heading / Text / Image buttons onto the white surfaces.
Just drag and let go.

  • easy
  • user-frendly
  • quick

How To Move Content?

Just click and drag them around. Content is taking up styles predefined in CSS so if you move text from main column to side column it will use styles defined for there.

How To Deal With Images

After draging image icon to the page click on it and select one or more images to load. Once you have image loaded you can click three times on it and change go through three edit modes: Resize, Crop and Frame. Try to pull selection handles to play a little.

What About Links?

Select text you want and in toolbar will appear link choices. Pick what you want and paste URL in the input box.

Undo/Redo

Undo and redo are pretty fast. Check them out.
